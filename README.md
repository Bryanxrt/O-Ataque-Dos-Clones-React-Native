# O-Ataque-Dos-Clones-React-Native# 🚗 Clone - CNH Digital App

Projeto desenvolvido como parte da **Aventura 1: O Ataque dos Clones** da turma 37 do SERRATEC.

## 📱 Sobre o Projeto

Clone visual da tela principal do aplicativo **CNH Digital** (app oficial do Governo Federal brasileiro), desenvolvido com **React Native** e **TypeScript**.

## 🖼️ Referência Utilizada

A imagem abaixo foi usada como referência para o clone:

> Tela Home do app CNH Digital, contendo os cards de navegação: Condutor, Veículos, Infrações e Educação.

## 🛠️ Tecnologias Utilizadas

- React Native
- TypeScript
- StyleSheet
- Componentes: `View`, `Text`, `TextInput`, `Image`

## ▶️ Como Rodar

```bash
# Instalar dependências
npm install

# Iniciar o projeto
npx expo start
```
